Easy Quran is a progressive web application. You can read on web page and install on your mobile device.

After installing without internet connection it can run offline.

Features

  • Easy to read
  • Easy to use
  • Scrolls top-to-bottom (Not left-to-right)
  • Verse number before verse (Not after)
  • Runs on web and mobile (Responsive)
  • Installable
  • Runs offline
  • Lightweight (About 2MB)
  • Lightning fast
  • Surah selection (* - has sajdah verse)
  • Juz selection
  • Page selection
  • Shortcuts to most read suras
  • Font family selection
  • Font size selection
  • Color selection
  • Background color selection
  • Multi language
  • Bookmark juz, page or even verse
  • Printable (Press Ctrl+P)

Icons

  • Quran navigation (Goto surah, juz, page)
  • Program info popup
  • Go to top
  • Go to bottom
  • Delete Bookmark
  • p 510 Page bookmark (Click to go to the page)
  • Settings (Select font, size, color, background, language)
  • p 510 Page number (Click to bookmark)
  • i Page info
  • Sajdah verse
  • Order by Sura Name
  • Order by Sura Number

While starting and changing settings it may take a bit time. Please be patient. Then scrolling is lightning fast!
